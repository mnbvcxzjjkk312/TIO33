<h1># TIO33</h1>
<h2>1. 프로젝트 명</h2>
 자바 스터디 게시판


<h2>2. 개발 목적 및 프로젝트 개요</h2>

 1) 개발 목적<br>
 개발 입문자를 위한 JAVA 프로그래밍 언어의 기본 개념 이해 및 질의 응답 게시판.<br>
 개발 입문자들이 낯설어 하는 코딩을 타자 연습과 같은 방식으로 적응도를 높여 주어 코드에 익숙하게 함.<br>
 학습한 JAVA 코드의 복습. 학습한 기능 구현의 복습 및 새로운 코드의 예습.
 

 2) 개발 인원 : 
 5명.

 3) 개발 기간 : 
 2019.11.28 - 2019.12.06 (7일)


<h2>3. 개발 환경</h2>
 1) 사용 언어 : JAVA<br>
 2) 사용 툴 : Eclipse, SQLDevelopter, MySQL Workbench, Netbeans
 
 
 <h2>3. 구현 기능 및 핵심 기술</h2>
  1) 주요 구현 기능<br>
     회원 가입<br>
     아이디 중복 확인<br>
     로그인<br>
     아이디 및 비밀번호 검색<br>
     글 등록 및 삭제 (CRUD)<br>
  2) 핵심 기술<br>
     JFrame UI 구현<br>
     Database 설계<br>
     JDBC를 이용한 Database 연동<br>
